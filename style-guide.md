# Front-end Style Guide

## Layout

The designs were created to the following widths:

- **Mobile**: 375px
- **Desktop**: 1440px

## Typography

### Body Copy

- **Font size (paragraph)**: 16px

### Font

- **Family**: [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans)
- **Weights**: 400, 500, 700
